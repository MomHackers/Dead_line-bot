# DeadLine Bot VK

Бот с дедлайнами для групп

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install project requirements.

```bash
pip install requirements.txt
```

## Usage

```bash
python vk-server.py
```
