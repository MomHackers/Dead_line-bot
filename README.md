# DeadLine Bot VK

Бот с дедлайнами для групп
Может создавать для вас дедлайны, напоминать расписание ближайших пар ...(в разработке) 

## Installation

Построен на асинхронной библиотеке VKBottle. Зависимости:
```bash
pip install requirements.txt
```

## Usage

```bash
python bot.py
```
